Style Your Wordpress Backend!

== Description ==

With this plugin you can add custom style to your wordpress backend. This first release will have the basic (cool!) editor. We'll add themes and utils soon!

This plugin has been created with a precise mission: give users a design experience diffent on wordpress.

It uses a JS library to make the editor cool, called [CodeMirror] (more info here: http://codemirror.net/) other stuff has been developed from me, Paolo Falomo


**Plugin Features**:

* Change your backend css
